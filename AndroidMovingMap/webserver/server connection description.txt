number  direction   (message)
1           ->      "hello"

3           ->      "sendPOI"
4           ->      "stopPOI"
5       <-          "POI:<JSON POI list>"
6           ->      "addOverlay:<POI TYPE>"
7           ->      "removeOverlay:<POI TYPE>"
8           ->      "addPoint:<POI>"
9           ->      "removePoint:<POI>"
10          ->      "close"

server on receiving 1 returns 2
server on receiving 3 starts periodic sending of number 5
server on receiving 4 stops periodic sending of number 5
server on receiving 6 will start including POI of type <POI TYPE> in number 5
server on receiving 7 will stop including POI of type <POI TYPE> in number 5
server on receiving 8 will add POI to global list of POI
server on receiving 9 will remove POI from global list of POI 
server on receiving 10 will close socket and delete preferences
